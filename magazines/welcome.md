# 시대영재 서현관에 오신 것을 환영합니다

서현고등학교 학생들이 함께 만들어가는 학습 자료 공유 플랫폼, **시대영재 서현관**에 오신 것을 환영합니다.

## 프로젝트 소개

시대영재 서현관은 서현고등학교 학생들이 직접 제작한 인강(인터넷 강의)과 학습 관련 매거진을 공유하는 플랫폼입니다.

### 주요 콘텐츠

- **해설 영상**: 시험 해설, 과목별 핵심 정리 영상
- **학습 매거진**: 공부 팁, 학습 전략, 과목별 심화 내용

## 매거진 작성 방법

매거진에 글을 추가하고 싶다면 다음 단계를 따라주세요:

1. `magazines/` 폴더에 새 마크다운 파일(`.md`)을 만듭니다
2. 마크다운 형식으로 글을 작성합니다
3. `js/magazines.js` 파일의 `magazinesData` 배열에 글 정보를 추가합니다:

```javascript
{
    slug: "파일이름",
    title: "글 제목",
    author: "작성자",
    date: "2025-06-01",
    summary: "글 요약",
    file: "magazines/파일이름.md"
}
```

4. 변경 사항을 커밋하고 푸시하면 사이트에 자동으로 반영됩니다.

## 영상 추가 방법

영상을 추가하려면:

1. `videos/` 폴더에 영상 파일을 넣습니다
2. `js/videos.js` 파일의 `videosData` 배열에 영상 정보를 추가합니다

> 함께 만들어가는 학습 공동체, 시대영재 서현관을 응원해주세요!
